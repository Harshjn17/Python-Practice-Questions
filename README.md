# Python Practice Questions

This repository contains a collection of Python practice problems and their solutions. It is organized by topics to help learners practice and improve their Python skills step by step.

---

## **Topics Covered**

1. **Basics**
   - Hello World programs
   - Input and Output
   - Variables and Data Types

2. **Loops**
   - Sum of Even and Odd numbers
   - Printing patterns
   - Factorial calculation

3. **Conditions**
   - Checking perfect numbers
   - Temperature conversion (Celsius ↔ Fahrenheit)
   - Weight conversion (Kg ↔ Lbs)

4. **Functions**
   - Creating reusable functions
   - Calculations using functions
   - Problem solving with function logic

5. **Miscellaneous**
   - Dollar to Rupees converter
   - Simple Interest & Compound Interest calculators
   - Other small practice programs
